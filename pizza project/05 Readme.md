# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

The **Pizza Sales Analysis Dashboard** is an end-to-end data analytics project designed to analyze pizza sales performance and generate actionable business insights. Using SQL for data analysis and Power BI for visualization, this project helps understand sales trends, customer purchasing behavior, and product performance through an interactive dashboard.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify best-selling and worst-selling pizzas.
- Monitor revenue, orders, and quantity sold.
- Track key business performance metrics.
- Create an interactive dashboard for business decision-making.

---

## 🛠️ Tools & Technologies

- SQL
- Power BI

## 📊 Dashboard Features

- Total Revenue
- Total Orders
- Total Pizza Sold
- Average Order Value
- Average Pizzas per Order
- Top 5 Pizzas by Revenue
- Top 5 Pizzas by Quantity
- Top 5 Pizzas by Total Orders
- Bottom 5 Pizzas by Revenue
- Bottom 5 Pizzas by Quantity
- Bottom 5 Pizzas by Total Orders
- Interactive Date Filter
- Pizza Category Filter

---

## 📈 Key Insights

- Identified the highest and lowest revenue-generating pizzas.
- Analyzed customer ordering patterns.
- Compared top and bottom-performing pizzas.
- Evaluated sales performance using key performance indicators (KPIs).
- Built an interactive dashboard for business analysis.
